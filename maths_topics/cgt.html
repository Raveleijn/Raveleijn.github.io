Combinatorial game theory

Combinatorial game theory is the study of combinatorial games, which are games which conform to the following conditions:

  1. Two players The game must be played with exactly two players.
  2. Turnbased The two players alternatingly take turns.
  3. Complete knowledge There is no hidden information for either player.
  4. Not based on chance Everything that happens in the game must not be based on chance.
  5. Finite The game must, if played well, always end.
  6. No Draws There must always be exactly one winner.
